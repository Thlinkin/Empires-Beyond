# Empires-Beyond
An AI experiment. With guidance, an attempt to make a entropic civilization simulator using only AI. I have done none of the code in this project, only guided the AI on what to do. The model used was ChatGPT 5.2 and I framed this project as a limit test for current AI models. Needless to say, this is very impressive and though provoking.
